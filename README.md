# Photo Clustering App

Цей проєкт реалізує **кластеризацію фото** на мобільному додатку за допомогою React Native та Expo.

## Що зроблено

- **Екран кластерів** – показує всі кластери фото з підрахунком кількості фото та фото з геолокацією.
- **Фільтри та режими кластеризації** – можна переключатись між кластеризацією по даті або по змісту, а також фільтрувати фото за наявністю геолокації.
- **Екран деталей кластеру** – показує назву, опис, місцезнаходження і сітку фото для конкретного кластеру.
- **Перегляд фото** – повноекранний режим перегляду з перегортанням, датою створення та геолокацією.
- **ML-класифікація фото** – автоматичне визначення вмісту фото через ML Kit, переклад міток і використання їх для кластеризації.
- **Оптимізоване завантаження фото** – батчами, з обробкою помилок і прогресом.
- **Повна підтримка дозволів** – запит доступу до галереї і обробка відмов.

## Чому це добре
- Дозволяє **швидко організувати велику кількість фото** на основі змісту або дати.
- Використовує **ML для автоматичної класифікації**, що зменшує ручну роботу.
- Підтримує **геолокаційні фільтри**, що зручно для фотоархівів.
- Реалізовано **стабільну та безпечну обробку фото**, навіть якщо частина файлів недоступна або пошкоджена.
- **Простий і зручний інтерфейс** з плавним перегортанням та повноекранним переглядом фото.

```bash
# Встановлюємо залежності
npm install

# Prebuild + pod install
npx expo prebuild && cd ios && pod install --repo-update && cd ..

# Запуск development build 
npx expo run:ios
